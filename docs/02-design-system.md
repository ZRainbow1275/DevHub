# DevHub - 设计系统

> 苏联构成主义（主）+ 瑞士秩序主义 + 数字构成主义 + 东方简约美学（副）

---

## 1. 设计哲学

### 1.1 核心理念

**「功能即美学」** — 每一个视觉元素都服务于功能，拒绝装饰性冗余。

| 风格来源 | 设计原则 | 在 DevHub 中的体现 |
|----------|----------|-------------------|
| **苏联构成主义** | 几何形态、对角线张力、红黑白三色 | 主视觉语言、强调色、动态指示 |
| **瑞士秩序主义** | 网格系统、无衬线字体、信息层级 | 布局结构、排版系统、数据展示 |
| **数字构成主义** | 像素感、科技线条、数据可视化 | 状态指示器、进度条、终端风格 |
| **东方简约美学** | 留白、克制、意境 | 呼吸空间、动效节奏、静谧基调 |

### 1.2 视觉张力

## 3. 排版系统

### 3.1 字体选择

```css
/* 主字体：几何无衬线，致敬包豪斯与瑞士风格 */
--font-primary: 'Inter', 'SF Pro Display', system-ui, sans-serif;

/* 等宽字体：终端日志、代码显示 */
--font-mono: 'JetBrains Mono', 'Fira Code', 'Consolas', monospace;

/* 中文字体：东方简约，清晰克制 */
--font-cjk: 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif;
```

### 3.2 字体层级

| 层级 | 尺寸 | 字重 | 用途 |
|------|------|------|------|
| Display | 32px | 700 | 页面标题 |
| Heading 1 | 24px | 600 | 区块标题 |
| Heading 2 | 18px | 600 | 卡片标题 |
| Body | 14px | 400 | 正文内容 |
| Caption | 12px | 400 | 辅助说明 |
| Mono | 13px | 400 | 日志、代码 |

### 3.3 行高与间距

```css
--line-height-tight: 1.25;    /* 标题 */
--line-height-normal: 1.5;    /* 正文 */
--line-height-relaxed: 1.75;  /* 长文本 */

--letter-spacing-tight: -0.02em;  /* 大标题 */
--letter-spacing-normal: 0;       /* 正文 */
--letter-spacing-wide: 0.05em;    /* 标签、按钮 */
```

## 6. 动效系统

### 6.1 时间曲线

```css
/* 东方美学：克制、自然的动效节奏 */

--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);     /* 主要过渡 */
--ease-in-out-sine: cubic-bezier(0.37, 0, 0.63, 1); /* 循环动画 */
--ease-spring: cubic-bezier(0.34, 1.56, 0.64, 1);   /* 弹性反馈 */

--duration-instant: 100ms;   /* 即时反馈 */
--duration-fast: 150ms;      /* 快速过渡 */
--duration-normal: 250ms;    /* 标准过渡 */
--duration-slow: 400ms;      /* 复杂动画 */
```

### 6.2 核心动效

| 场景 | 动效 | 参数 |
|------|------|------|
| 按钮悬停 | 背景淡入 | 150ms ease-out |
| 按钮点击 | 缩放 + 涟漪 | 100ms spring |
| 卡片展开 | 高度过渡 | 250ms ease-out-expo |
| 状态变化 | 颜色过渡 | 200ms ease-out |
| 日志滚动 | 平滑滚动 | 自然阻尼 |
| 通知弹出 | 从右滑入 | 300ms ease-out-expo |

### 6.3 状态指示器

```css
/* 运行中状态：构成主义脉冲 */
@keyframes pulse-constructivist {
  0%, 100% {
    opacity: 1;
    transform: scale(1);
  }
  50% {
    opacity: 0.7;
    transform: scale(1.05);
  }
}

/* 对角线扫描（加载状态） */
@keyframes diagonal-scan {
  0% {
    background-position: -200% 0;
  }
  100% {
    background-position: 200% 0;
  }
}
```

### 7.1 设计原则

- **几何构成**: 基于圆形、方形、三角形

## 9. 响应式考量

虽然是桌面应用，仍需考虑不同窗口尺寸：

| 宽度 | 布局调整 |
|------|----------|
| < 800px | 侧边栏收起为图标模式 |
| 800-1200px | 标准双栏布局 |
| > 1200px | 可选三栏（项目列表 + 详情 + 日志） |

---

## 10. 无障碍设计

- **对比度**: 所有文字对比度 ≥ 4.5:1
- **焦点指示**: 清晰的键盘焦点环
- **屏幕阅读器**: 完整的 ARIA 标签
- **快捷键**: 所有核心操作支持键盘快捷键
